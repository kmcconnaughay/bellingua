# Nomen

## Grundlegende Vokabeln

- der Browser / -
- der Code
- das Programm / -e
- der Texteditor / -en
- das Terminal / -s

## Komputersprachen

- das JavaScript
- die Hypertext-Auszeichnungssprache / -n (HTML)
- kaskadierte Stylesheets {pl} (CSS)

## Website Vokabeln

- die CSS-Klasse / -n
- das HTML-Element / -e
- der Hyperlink / -s
- die Website / Webseiten

# Verben

- eine Website hosten (hostet, hostete, hat gehostet)
- programmieren (programmiert, programmierte, hat programmiert)
- etw. rendern (rendert, renderte, hat gerendert)
- etw. veröffentlichen (veröffentlicht, veröffentlichte, hat veröffentlicht)
